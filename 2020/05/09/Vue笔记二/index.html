Vue笔记二

Share

插值操作

文本

  • Mustache语法（双大括号）
    最常见的就是“Mustache”语法，即双大括号,可以直接写变量，也可写简单的表达式或空格
  • v-once
    使用v-once可以一次性插值，数据改变时插值内容也不会更新
  • v-text
    不常用，会覆盖掉h2中的内容
    <h2 v-text="message">123</h2>（结果会显示message的内容，覆盖123）

v-html

为了输出html代码，需要使用v-html指令
<h2 v-html="url"></h2>
url:<a href="http://www.baidu.com">百度一下</a>
这里h2标签就会变成百度一下的连接，而不是显示纯文本。

注：站点上动态渲染的任意 HTML 可能会非常危险，因为它很容易导致 XSS 攻击。请只对可信内容使用 HTML 插值，绝不要对用户提供的内容使用插值。

v-pre

直接将h2里面的内容原封不动显示出来

v-cloak

这个指令保持在元素上直到关联实例结束编译。和 CSS 如 [v-cloak] { display: none } 一起用时，这个指令可以隐藏未编译的 Mustache 标签直到实例准备完毕。

结合上面的CSS，在Vue解析到这段代码之前，h2中有v-cloak，h2就会隐藏起来；在Vue解析之后，div中没有v-cloak，h2就会显示出来。

动态绑定属性

v-bind

动态的绑定一个或多个属性，或一个组件prop到表达式。
<img v-bind:src="imgUrl">
可以简写成：
<img :src="imgUrl">

v-bind动态绑定class（对象语法）

<div v-bind:class="{ active: isActive }"></div>
active为类名，isActive为布尔值，当isActive值为true时就会给div添加active这个class；可以在对象中传入多个字段来动态切换多个class，只需要改变其布尔值。

绑定的数据对象不必内联定义在模板里：
<div v-bind:class="classObject"></div>

这样就会给div添加了active属性而没有添加line属性。

v-bind动态绑定class（数组语法）

<div v-bind:class="[active,line]"></div>

v-bind动态绑定style（对象语法）

v-bind:style 的对象语法十分直观，看着非常像 CSS，但其实是一个 JavaScript 对象。
CSS属性名用驼峰式或者短横线分割（要加单引号括起来）。
<div v-bind:style="styleObject"></div>

v-bind动态绑定style（数组语法）

<div v-bind:style="[baseStyles, overridingStyles]"></div>
