javascript初识练习题（一）

Share

一、如何区分HTML和HTML5?

  • HTML4是用 <! DOCTYPE html PUBLIC “-//W3C//DTD XHTML 4.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd”> 这种的声明。
  • HTML5是用<!DOCTYPE>声明。
  • HTML5<canvas>标签代替flash。
  • HTML5新增<header><footer><section><article><menu><figure><audio><video>等标签。
  • HTML5删除了<b><font><frame><center><big>标签。
  • HTML5对<form>和 <forminput> 标签进行了大量修改，添加了很多新的属性，也修改了很多属性。
    个人小结：首先就是看文件开头的声明区别，然后寻找新增的语义化标签。

二、为什么HTML和HTML5的声明方式不同/背后的原理是什么？

  • HTML4中，<!DOCTYPE>声明引用DTD（文档类型声明），因为HTML4基于SGLM（标准通用标记语言）。
  • HTML5不是SGLM子集，不需引用DTD。
    个人小结：本质上都是为了保证浏览器正确渲染内容，HTML5的doctype 比以前更短，更简单，使的它更容易被记住并且减少必须下载的字节数。

三、JS这门语言由哪些部分组成，不同的部分都有什么作用？

由EMCAscript、DOM、BOM组成，其中：

  • EMCAscript：提供核心语言功能,是核心，规定了这们语言的书写规范；
  • DOM：提供访问和操作网页内容的方法和接口,(document object model 简称DOM 文档对象模型)；
  • BOM: 提供与浏览器交互的方法和接口,从根本上讲BOM只处理浏览器窗口和框架,(browser object model 简称 BOM 浏览器对象模型)。

四、如何判断JS的数据类型？

最简单的检测数据类型的方式是通过typeof的方式来检测

五、Javascript有几种数据类型？

  • 基本数据类型：String、number、boolean、null、undefined。
  • 引用数据类型：Object。

六、typeof(null) 输出的是什么？

是Object而不是null，在javascript里，null是一个空指针。
个人小结：typeof返回的结果是一个字符串；里面包含了对应的数据类型，两次和两次以上的连续typeof结果肯定是string。

七、Javascript代码中的”use strict”是什么意思，使用它和不使用它有什么区别？

是进入严格模式的方式，用于解决JS本身的语法问题：

  • 消除Javascript语法的一些不合理、不严谨之处，减少一些怪异行为;
  • 消除代码运行的一些不安全之处，保证代码运行的安全；
  • 提高编译器效率，增加运行速度；
  • 为未来新版本的Javascript做好铺垫。

八、引入样式表CSS/JS的方式有几种？分别是什么，优先级有什么不同？

分为行内式，嵌入式，外联式。
优先级：行内式>嵌入式>外联式。
个人小结：养成使用外联式的习惯，其他两种形式也要会使用。

九、noscript标签是用来做什么的？

noscript 元素用来定义在脚本未被执行时的替代内容（文本），也可以用在检测浏览器是否支持脚本，若不支持脚本则可以显示noscript标签里的innerText。
