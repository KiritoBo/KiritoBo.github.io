一、不同数据类型之间的转换

  • 转换成字符串toString(),String(),eval(),其中String()可用于转换null和undefined,计算字符串表达式的值并以数值形式返回；
    快速转换字符串的方式：在数据后面加上+””
  • 转换成数字Number(),parseInt(),parseFloat,+”77”(隐式)，”22”-0（隐式）
  • 转换成布尔值boolean()，!，!!，
  • 小结：转换后为false的值有false 本身、空字符串、0、NaN、null、undefined。

    二、null和undefined区别和联系

  • null和undefined都是表示没有的，不存在的值；他们两个在进行逻辑转换时候都是false；这两个值进行比较是true；
  • null表示空引用，它是object类型，undefined表示未定义，它是undefined类型；
  • null是object类型，但不是object的实例；用instanceof为false；
  • 一个变量未定义，或者变量定义里未赋值；则是undefined；

    三、在js中，{}、[]、/^$/分别代表什么意思？

    {}是对象，[]是数组，/^$/是正则

    四、NaN是什么意思？这个值有什么特点？

  • Not a Number 不是数值但是NaN属于数值类型
  • 任何涉及NaN的操作都将返回NaN
  • NaN与任何数值都不相等包括其自身

    八、请写出下面式子的结果

  • null instanceof Object false
    instanceof是检测检测某一个实例是否属于这个类，null是object类型，但不是object的实例；用instanceof为false；
  • null===undefined false
    null表示空引用，它是object类型，undefined表示未定义，它是undefined类型；
  • null==undefined true
    因为undefined派生自null，所以undefined和null做数据类型比较的时候的true；
  • NaN==NaN false
    NaN和任何数值都不想等；
  • false==0 true
    ==将false转换为数值0进行比较所以相等；
  • false==-1 flase
    false转换成数字类型为0 不等于-1；
  • “”==0 true
    空串转换成数字类型为0；
  • []==0 true
    空数组转换成数字类型为0；
  • [] == [] false
    因为数组是对象类型，他们的地址不同，所以是false；
  • !![] true
    !! 是将其他的数据类型取反再取反，结果是将数据转换为布尔类型，相当于 Boolean()；对象类型布尔值为true；
  • false==null false
    null 和 undefined 和其他类型比较都是 false；（这两个方法没有 toString 方法）
  • false==”” true
    布尔值和字符串比较时都转换成数字在比较，false和空串转换数字类型都是0；
  • false==” “ true
    布尔值和字符串比较时都转换成数字在比较；
  • 0.1+0.2==0.3 false
    浮点数之间的运算不准确，需要转成整数，然后再运算，运算后再转回对应的小数，实际上0.1+0.2=0.30000000000000004；
  • “5”==5 true
    字符串和数字比较时两边都转换为数字，两边相等；
  • 补充知识：==：运算符称作相等百，用来检测两个操作数是否相等，这里的相等定义的非常宽松，可以允许进行类型转换
    ===：用来检测两个操作数是否度严格相等
