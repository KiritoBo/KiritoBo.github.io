一、不同数据类型之间的转换

  • 转换成字符串toString(),String(),eval(),其中String()可用于转换null和undefined,计算字符串表达式的值并以数值形式返回；
    快速转换字符串的方式：在数据后面加上+””
  • 转换成数字Number(),parseInt(),parseFloat,+”77”(隐式)，”22”-0（隐式）
  • 转换成布尔值boolean()，!，!!，
  • 小结：转换后为false的值有false 本身、空字符串、0、NaN、null、undefined。

    二、null和undefined区别和联系

  • null和undefined都是表示没有的，不存在的值；他们两个在进行逻辑转换时候都是false；这两个值进行比较是true；
  • null表示空引用，它是object类型，undefined表示未定义，它是undefined类型；
  • null是object类型，但不是object的实例；用instanceof为false；
  • 一个变量未定义，或者变量定义里未赋值；则是undefined；

    三、在js中，{}、[]、/^$/分别代表什么意思？

    {}是对象，[]是数组，/^$/是正则

    四、NaN是什么意思？这个值有什么特点？

  • Not a Number 不是数值但是NaN属于数值类型
  • 任何涉及NaN的操作都将返回NaN
  • NaN与任何数值都不相等包括其自身

    五、请问alert(Number(“08”))；输出的结果是什么？

    结果是8 先将”08”转换成8然后输出为数字8

    六、如何让alert弹出框的信息换行？

    使用\n

    七、var str=true+11+null+9+undefined+”zhuanbang”+false+null+9+[],str的值是多少？为什么？

    NaNzhuanbangfalsenull9

    八、请写出下面式子的结果

  • null instanceof Object false
    instanceof是检测检测某一个实例是否属于这个类，null是object类型，但不是object的实例；用instanceof为false；
  • null===undefined false
    null表示空引用，它是object类型，undefined表示未定义，它是undefined类型；
  • null==undefined true
    因为undefined派生自null，所以undefined和null做数据类型比较的时候的true；
  • NaN==NaN false
    NaN和任何数值都不想等；
  • false==0 true
    ==将false转换为数值0进行比较所以相等；
  • false==-1 flase
    false转换成数字类型为0 不等于-1；
  • “”==0 true
    空串转换成数字类型为0；
  • []==0 true
    空数组转换成数字类型为0；
  • [] == [] false
    因为数组是对象类型，他们的地址不同，所以是false；
  • !![] true
    !! 是将其他的数据类型取反再取反，结果是将数据转换为布尔类型，相当于 Boolean()；对象类型布尔值为true；
  • false==null false
    null 和 undefined 和其他类型比较都是 false；（这两个方法没有 toString 方法）
  • false==”” true
    布尔值和字符串比较时都转换成数字在比较，false和空串转换数字类型都是0；
  • false==” “ true
    布尔值和字符串比较时都转换成数字在比较；
  • 0.1+0.2==0.3 false
    浮点数之间的运算不准确，需要转成整数，然后再运算，运算后再转回对应的小数，实际上0.1+0.2=0.30000000000000004；
  • “5”==5 true
    字符串和数字比较时两边都转换为数字，两边相等；
  • 补充知识：==：运算符称作相等百，用来检测两个操作数是否相等，这里的相等定义的非常宽松，可以允许进行类型转换
    ===：用来检测两个操作数是否度严格相等

javascript初识练习题（一）

Share

一、如何区分HTML和HTML5?

  • HTML4是用 <! DOCTYPE html PUBLIC “-//W3C//DTD XHTML 4.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd”> 这种的声明。
  • HTML5是用<!DOCTYPE>声明。
  • HTML5<canvas>标签代替flash。
  • HTML5新增<header><footer><section><article><menu><figure><audio><video>等标签。
  • HTML5删除了<b><font><frame><center><big>标签。
  • HTML5对<form>和 <forminput> 标签进行了大量修改，添加了很多新的属性，也修改了很多属性。
    个人小结：首先就是看文件开头的声明区别，然后寻找新增的语义化标签。

二、为什么HTML和HTML5的声明方式不同/背后的原理是什么？

  • HTML4中，<!DOCTYPE>声明引用DTD（文档类型声明），因为HTML4基于SGLM（标准通用标记语言）。
  • HTML5不是SGLM子集，不需引用DTD。
    个人小结：本质上都是为了保证浏览器正确渲染内容，HTML5的doctype 比以前更短，更简单，使的它更容易被记住并且减少必须下载的字节数。

三、JS这门语言由哪些部分组成，不同的部分都有什么作用？

由EMCAscript、DOM、BOM组成，其中：

  • EMCAscript：提供核心语言功能,是核心，规定了这们语言的书写规范；
  • DOM：提供访问和操作网页内容的方法和接口,(document object model 简称DOM 文档对象模型)；
  • BOM: 提供与浏览器交互的方法和接口,从根本上讲BOM只处理浏览器窗口和框架,(browser object model 简称 BOM 浏览器对象模型)。

四、如何判断JS的数据类型？

最简单的检测数据类型的方式是通过typeof的方式来检测

五、Javascript有几种数据类型？

  • 基本数据类型：String、number、boolean、null、undefined。
  • 引用数据类型：Object。

六、typeof(null) 输出的是什么？

是Object而不是null，在javascript里，null是一个空指针。
个人小结：typeof返回的结果是一个字符串；里面包含了对应的数据类型，两次和两次以上的连续typeof结果肯定是string。

七、Javascript代码中的”use strict”是什么意思，使用它和不使用它有什么区别？

是进入严格模式的方式，用于解决JS本身的语法问题：

  • 消除Javascript语法的一些不合理、不严谨之处，减少一些怪异行为;
  • 消除代码运行的一些不安全之处，保证代码运行的安全；
  • 提高编译器效率，增加运行速度；
  • 为未来新版本的Javascript做好铺垫。

八、引入样式表CSS/JS的方式有几种？分别是什么，优先级有什么不同？

分为行内式，嵌入式，外联式。
优先级：行内式>嵌入式>外联式。
个人小结：养成使用外联式的习惯，其他两种形式也要会使用。

九、noscript标签是用来做什么的？

noscript 元素用来定义在脚本未被执行时的替代内容（文本），也可以用在检测浏览器是否支持脚本，若不支持脚本则可以显示noscript标签里的innerText。
